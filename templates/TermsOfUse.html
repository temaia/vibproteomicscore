Terms of Use


  1. Training
    • Before getting access to any specific instrument, users should receive training by the facility staff before initiating equipment operation by themselves. The user may only sign up for instruments for which they have been formally trained on or authorized to use after demonstrated expertise. Training on any instruments are conducted by facility staff ONLY.

  2. Reservations and Cancellations
    • All reservations should be made by using the online-reservation system. If a user is found to use an instrument during an unscheduled time, or significantly beyond the scheduled time without notifying the core facility staff, the userâ€™s lab will be responsible for instrument charges incurred, the PI will be notified. Cancellation or modification of a reserved time slot is required within 24 hours before the scheduled time slot. Cancellation during the scheduled time slot is not possible by the user, and has to be discussed with the core facility. The user may be financially responsible for any such last minute changes. In case of a no-show or no advanced notification, the user will be charged for the reserved time slot.

  3. Responsibility for Equipment Misuse
    • If equipment is damaged due to improper and negligent use, the P.I. is responsible for part or all of repair costs.

  4. Data storage
    • Raw data are temporarily stored and backed up on the server of the core facility. All raw data will be deleted without warning after 2 years. In case an extension to this period is required, the facility manager should be informed timely.

  5. Acknowledgements
    • In case data collected by use of the Bio-Imaging Core equipment are used in publications, the Bio-Imaging Core needs to be acknowledged.
